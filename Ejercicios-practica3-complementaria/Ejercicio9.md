# Ejercicio 9

## Buscar en la RFC de DHCP la utilidad del mensaje DHCPInform.

Según la RFC 2131, el mensaje DHCPINFORM sirve para que un cliente que ya posee una dirección IP (configurada manualmente o por otros medios) pueda solicitar al servidor parámetros de configuración adicionales (como direcciones de servidores DNS, nombre de dominio, servidores WINS, servidor SMTP, etc.) sin solicitar una asignación de dirección IP ni establecer tiempo de concesión (lease).
